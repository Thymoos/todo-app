This project is a ToDo App created for Frontend Mentor: 
https://www.frontendmentor.io/challenges/todo-app-Su1_KokOW/hub/todo-app-4ntlE1q5I

- Complete or delete tasks

- Light / Dark theme
- Works on mobile devices
